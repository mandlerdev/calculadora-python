# calculadora-python
Calculadora em Python com funções, validação de entrada e operações matemáticas.

Projeto simples de calculadora desenvolvido em Python para praticar lógica de programação.

Funcionalidades:
Soma
Subtração
Multiplicação
Divisão
Loop para múltiplos cálculos

Tecnologias - Python

Objetivo - Praticar estruturas condicionais, loops e entrada de dados.
